# gti

## What is this?

You know how `sl` exists for when you type `ls` wrong? This is that, but for `git`.

Except instead of a train, you get a **Volkswagen Golf GTI Mk1** racing across your terminal.

## Why?

Because if you're going to make a typo, you might as well watch a hot hatch do burnouts.

## Installation

```bash
chmod +x gti
sudo cp gti /usr/local/bin/
```

Then just keep typing `gti` instead of `git` like the dyslexic developer you were born to be.

## Features

- `gti status` - Check your GTI's vital stats (not your repo, your *car*)
- `gti push` - Watch the GTI zoom off to the right (hopefully to production)
- `gti pull` - Watch the GTI reverse back in (fetching those sweet, sweet commits)
- `gti commit` - *click* Ignition sequence start
- `gti <anything else>` - Just admire the car, you've earned it
